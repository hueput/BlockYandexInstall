# BlockYandexInstall
Скрипт создаёт папки Yandex в типичном их расположении и блокирует возможность записи в них, тем самым блокируя любую возможность установки вируса "Яндекс"

## Как запустить:
  1. Скачайте скрипт на свой ПК
  2. Перенесите его в `C:\Users\%username%`
  3. Откройте PowerShell от имени администратора
  4. Используйте команду `.\BlockYandex.ps1`

Работа проверена на Windows 11 Pro, 06.03.2026: установка прерывается на 40% загрузки
